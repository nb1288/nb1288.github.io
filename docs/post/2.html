申請過程

1.打開App Store/Google Play，搜索下載RedteaGo App

2.打開App，點擊右下角的Me

3.點擊Sign up

4.輸入郵箱

5.創建一個密碼

6.輸入郵箱收到的6位數認證碼

7.註冊完成後點擊右下角的Me，然後點擊Promo Code

8.輸入NB120001，然後點擊Redeem

9.回到剛才的頁面會發現你收到了US$3.00餘額

10.App主頁搜索China，購買1GB 31天數據包即可
